# randommapurt plugin
# Plugin for B3 (www.bigbrotherbot.net)
# www.ptitbigorneau.fr

randommapurt B3 plugin (v1.3) for Urban Terror

Requirements
------------

* BigBortherBot(3) >= version 1.10

Installation:
-------------

1. Copy the 'randommapurt' folder into 'b3/extplugins' and 'randommapurt.ini' file into '/b3/extplugins/conf'.

2. Open your B3.ini or b3.xml file (default in b3/conf) and add the next line in the [plugins] section of the file:
    for b3.xml
        <plugin name="randommapurt" config="@b3/extplugins/conf/randommapurt.ini"/>
    for b3.ini
        randommapurt: @b3/extplugins/conf/randommapurt.ini
